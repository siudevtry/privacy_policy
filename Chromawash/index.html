Chromawash

Privacy Policy

This Privacy Policy explains how the Chromawash browser extension handles information when you install and use it.

Effective date: July 13, 2026

Privacy summary Chromawash does not collect personal information, does not track browsing activity, does not display advertising, and stores only extension preferences locally in the browser.

1. About Chromawash

Chromawash is a browser extension that overlays a whole-page glowing color wash with rhythmic pulses, slow gradient shifts, and soft luminous edges. Users can activate the effect and customize its appearance through the extension's toolbar popup.

The extension does not require an account, login, subscription, or connection to a developer-operated service.

2. Information We Do Not Collect

Chromawash does not collect, store, process, or transmit personal information. In particular, the extension does not collect:

  • Names, email addresses, account details, or contact information
  • Browsing history, visited URLs, bookmarks, or search queries
  • Website content, form entries, passwords, or payment information
  • IP addresses, device identifiers, location data, or demographic information
  • Usage analytics, advertising identifiers, or behavioral tracking data

3. Local Extension Settings

Chromawash uses browser storage only to remember settings selected by the user, such as whether the effect is enabled and the available color, intensity, timing, or visual preferences.

These settings are stored locally in the browser's extension storage. They are not uploaded to the developer, transmitted to an external server, sold, or shared with third parties.

Local settings may be removed by resetting the extension, clearing browser extension data, or uninstalling Chromawash.

4. Browser Permissions

Chromawash requests only the permissions needed to provide its declared visual effect:

  • activeTab — allows the extension to operate on the current active tab only after the user explicitly activates it.
  • scripting — allows the extension to insert and run the code required to display the color wash effect.
  • storage — allows the extension to save user-selected preferences locally in the browser.

These permissions are not used to monitor browsing activity, collect website content, or access information unrelated to the extension's visible functionality.

5. Advertising, Analytics, and Tracking

Chromawash does not contain advertisements and does not use analytics services, tracking pixels, cookies, fingerprinting, telemetry, or behavioral profiling.

The extension does not create advertising profiles and does not share information with advertisers, analytics providers, or data brokers.

6. Remote Code and External Services

Chromawash does not use remote code. All code required for the extension's functionality is included in the installed extension package.

The extension does not depend on third-party advertising, analytics, or tracking services. The browser or extension marketplace used to install the extension may process information under its own privacy policy.

7. Data Sharing and Sale

Chromawash does not sell, rent, license, disclose, or share personal data. Because the extension does not collect personal data, no personal user database is maintained by the developer.

8. Data Security and Retention

The extension is designed to minimize privacy risk by avoiding the collection and transmission of personal information. User preferences remain in the browser's local extension storage.

Any locally retained settings can generally be removed by resetting the extension, clearing browser extension data, or uninstalling Chromawash.

9. Children's Privacy

Chromawash does not knowingly collect personal information from children or from users of any age. The extension does not provide account registration, messaging, or social networking features.

10. Changes to This Privacy Policy

This Privacy Policy may be updated if the extension's features, permissions, or data practices change. When an update is made, the effective date at the top of this page will be revised.

Any future version that introduces new data handling practices will require this policy to be updated accordingly.
